# Module 3: Linux Process Management


## About This Module

This module focuses on Linux process management, including process monitoring, 
resource usage analysis, and process control operations commonly performed on Linux servers.


## What Is Covered

- Listing running processes
- Monitoring CPU and memory usage
- Identifying high resource processes
- Stopping running processes
- Force killing unresponsive processes
- Adjusting process priority
- Verifying process termination


## Folder Structure

- **README.md** - Module overview and learning scope 
- **practice.md** - Hands-on commands and execution steps
- **incident.md** - Simulated incidents and basic troubleshooting scenarios
- **screenshots/** - Folder containing screenshots the instructions and scenarios


## Key Topics

- Process and PID concepts 
- CPU and memory monitoring 
- High resource process identification 
- Process termination and signals 
- Process priority management 


## Outcome

After completing this module, the learner is able to monitor running processes, 
identify resource-related issues, and manage process execution in a Linux environment.
